JAVASCRIPT란?

-웹페이지와 상호작용하도록 만들어진 스크립트 언어로, 컴파일이 필요 없으며, HTML 웹 페이지를 통해 스크립트를 삽입하면 웹 브라우저를 통해서 동작하게 됩니다.

-ECMA-262에서 정의하는 ECMAScript의 색심 기능을 제공하며 있으며, 문서 객체 모델(DOM)을 통해 웹 페이지 콘텐츠를 조작하는 메서드와 인터페이스를 제공하고, 브라우저 객체 모델(BOM)을 통해 브라우저와 상호작용하는 메서드와 인터페이스를 제공합니다.

-웹 서버가 아닌 클라이언트 컴퓨터에 설치된 브라우저에서 실행되므로, 클라이언트 측 스크립트 언어라고 불립니다.

JSP장점?

1.배우기 쉽습니다.

2.컴파일,링크과정이 필요 없으므로 즉시 실행이 가능하고 그에 따라서 디버깅이 간편합니다

3.대부분의 웹 브라우저는 자바스크립트를 지원하므로 브라우저가 설치된 곳에서는 HTML처럼 언제든 볼 수 있습니다.

4.컴퓨터의 OS에 무관하게 프로그램이 수행됩니다.

4.웹서버에 주는 부담이 적습니다.
