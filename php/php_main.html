PHP란?
웹에서 사용하기 위해 설계된 서버 스크립트 언어로, 프로그램이 실행될 때 마다 프로그램의 코드가 실시간으로 실행되므로, 항상 똑같은 페이지(정적정보)가 아닌 항상 바뀌는 페이지(동적 정보)를 볼 수 있고, 웹사이트와 방문자 간의 상호소통을 할 수도 있습니다.
PHP장점?
1. 배우기 쉽습니다. 2. 가볍기 때문에 값싼 서버로 고효율적으로 활용할 수 있습니다. 3. 웹 기능을 지원하는 다양한 내장 라이브러리가 있습니다. 4. 데이터 표준형식인 ODBC(Open Database Connectivity Standard)도 지원합니다.
